Selected Projects
----

### DeepMaze

* What it is: A maze exploring game developed by Godot Engine.
* My Role: Programmer, Suggestion of Game Design
* What I implemented: Game loop, ability/skill system, UI implementation,
   game infrastructure system
* Problems Solved: Difference between UI in Figma and Godot,
   maintainability of game system, extendability of game loop.
* Public repo link: https://github.com/OzelotVanilla/deepmaze__public

### CattyPiano

* What it is: Browser-based piano/rhythm game
* My Role: Programmer, Designer, Music Composer
* Problems Solved: UX problems such as delay or strange feedback of input
* Repo link: https://github.com/OzelotVanilla/CattyPiano

### jjdev

* What it is: A CLI tool to let user watch `jj log` simutaneously
   while using [jj-vcs](https://github.com/jj-vcs/jj) in terminal.
* My Role: Programmer
* What I implemented: A tool opening a panel showing jj's log tree
   at right side, watch the changing of jj's state,
   and updating it when jj working states change.
* Problems Solved: Running `jj log` frequently to see commit change ID,
   which is commonly used in most `jj` command to refer to a commit.
* Repo link: https://github.com/OzelotVanilla/jjdev
